    Create `bigbangsim/presentation/camera_controller.py` with:

    1. **Pure Catmull-Rom function:**
       ```python
       def catmull_rom(p0: float, p1: float, p2: float, p3: float, t: float) -> float:
       ```
       Standard Catmull-Rom interpolation between p1 and p2. Works on scalar floats (applied per-component to azimuth, elevation, radius, fov individually). Formula:
       ```
       0.5 * ((2*p1) + (-p0+p2)*t + (2*p0 - 5*p1 + 4*p2 - p3)*t^2 + (-p0 + 3*p1 - 3*p2 + p3)*t^3)
       ```

    2. **CameraKeyframe frozen dataclass:**
       ```python
       @dataclass(frozen=True)
       class CameraKeyframe:
           azimuth: float       # Horizontal angle (degrees)
           elevation: float     # Vertical angle (degrees)
           radius: float        # Distance from target
           target_x: float      # Target X (use scalars, not glm.vec3, for testability)
           target_y: float      # Target Y
           target_z: float      # Target Z
           fov: float           # Field of view (degrees)
       ```

    3. **ERA_KEYFRAMES dict** mapping era_index (0-10) to a list of 1-2 CameraKeyframe entries. Design cinematic camera positions for each era:
       - Era 0 (Planck): Close-in, wide FOV (70), looking slightly down. azimuth=0, elevation=-15, radius=6.0
       - Era 1 (GUT): Slight orbit. azimuth=30, elevation=-20, radius=7.0, fov=65
       - Era 2 (Inflation): Pull back dramatically. azimuth=60, elevation=-10, radius=12.0, fov=75
       - Era 3 (QGP): Orbit around. azimuth=100, elevation=-15, radius=10.0, fov=65
       - Era 4 (Hadron): Continue orbit. azimuth=140, elevation=-25, radius=9.0, fov=60
       - Era 5 (Nucleosynthesis): Steady observation. azimuth=170, elevation=-20, radius=10.0, fov=60
       - Era 6 (Recombination): Pull back for CMB release. azimuth=200, elevation=-10, radius=14.0, fov=65
       - Era 7 (Dark Ages): Far out, narrow FOV. azimuth=230, elevation=-5, radius=20.0, fov=55
       - Era 8 (First Stars): Zoom back in. azimuth=260, elevation=-25, radius=15.0, fov=60
       - Era 9 (Galaxy Formation): Wide pan. azimuth=300, elevation=-20, radius=18.0, fov=55
       - Era 10 (LSS): Panoramic finale. azimuth=350, elevation=-30, radius=25.0, fov=50
       All targets at (0, 0, 0) for now (particles are centered at origin).

    4. **CinematicCameraController class:**
       ```python
       class CinematicCameraController:
           def __init__(self, camera: DampedOrbitCamera,
                        keyframes: dict[int, list[CameraKeyframe]] | None = None,
                        blend_back_duration: float = 1.5):
       ```
       - Stores reference to camera, keyframes (default ERA_KEYFRAMES), blend state.
       - `auto_mode: bool = True`
       - `blend_back_timer: float = 0.0`
       - `_saved_azimuth, _saved_elevation, _saved_radius, _saved_fov`: stored when entering free mode, used for blend-back source when re-entering auto.

       - `evaluate_path(self, era_index: int, era_progress: float) -> CameraKeyframe`:
         Evaluates the camera position at the given era and progress. For eras with a single keyframe, interpolate between adjacent eras' keyframes using Catmull-Rom:
         - Get keyframes for eras [era_index-1, era_index, era_index+1, era_index+2] (clamp at boundaries by repeating first/last era's keyframe).
         - Apply catmull_rom per component (azimuth, elevation, radius, fov, target_x/y/z) with t=era_progress.
         Returns a CameraKeyframe with interpolated values.

       - `update(self, dt: float, era_index: int, era_progress: float)`:
         If NOT in auto_mode, return immediately (camera is under user control).
         If in auto_mode:
           - Evaluate the scripted position via evaluate_path(era_index, era_progress).
           - If blend_back_timer > 0: smoothstep-blend between saved free-orbit state and scripted position. Decrement blend_back_timer by dt.
             alpha = 1.0 - (blend_back_timer / blend_back_duration)
             alpha = alpha * alpha * (3.0 - 2.0 * alpha)  # smoothstep
           - Else: directly apply scripted position.
           - Apply to camera: set camera.azimuth, camera.elevation, camera.radius, camera.fov, camera.target. Also zero out camera velocity fields (_vel_azimuth, _vel_elevation, _vel_zoom) to prevent damping interference.

       - `toggle_mode(self)`:
         If switching TO free mode (auto_mode was True):
           - Save current camera state (_saved_azimuth = camera.azimuth, etc.)
           - Set auto_mode = False
         If switching TO auto mode (auto_mode was False):
           - Save current free-orbit camera state as blend source
           - Set blend_back_timer = blend_back_duration
           - Set auto_mode = True

       - `@property is_auto(self) -> bool`: Return auto_mode.

    Anti-patterns to avoid (from research):
    - Do NOT bypass the camera's property interface -- set azimuth/elevation/radius/target directly, don't try to set position.
    - Do NOT use glm.vec3 in CameraKeyframe -- use scalar fields for easy serialization and testing.
    - Zero velocity fields when in auto mode to prevent damping from fighting the scripted position.
