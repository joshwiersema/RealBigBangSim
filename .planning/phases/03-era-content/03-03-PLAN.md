    **era_transition.py -- New EraTransitionManager class:**
    ```python
    class EraTransitionManager:
        """FBO-based crossfade between adjacent cosmological eras (RNDR-04).

        During era transitions, renders the scene twice (outgoing + incoming era),
        then composites with a blend factor that ramps from 0.0 to 1.0 over the
        transition duration.
        """
        def __init__(self, ctx: moderngl.Context, width: int, height: int):
            # Transition FBO (same RGBA16F pattern as PostProcessingPipeline)
            self.ctx = ctx
            self.transition_texture = ctx.texture((width, height), 4, dtype="f2")
            self.transition_texture.filter = (moderngl.LINEAR, moderngl.LINEAR)
            self.transition_fbo = ctx.framebuffer(
                color_attachments=[self.transition_texture]
            )

            # Composite shader: fullscreen.vert + era_crossfade.frag
            fs_vert = load_shader_source("postprocess/fullscreen.vert")
            crossfade_frag = load_shader_source("postprocess/era_crossfade.frag")
            self.composite_prog = ctx.program(
                vertex_shader=fs_vert, fragment_shader=crossfade_frag
            )
            self.quad = geometry.quad_fs()

            # Transition state
            self.in_transition = False
            self.blend_factor = 0.0
            self.outgoing_era = 0
            self.incoming_era = 0
            self.transition_elapsed = 0.0
            self.transition_duration = 2.0  # Updated from EraVisualConfig

        def check_transition(self, current_era: int, era_progress: float,
                            transition_seconds: float) -> None:
            """Check if we should start/continue/end a transition.

            Start transition when era changes (current_era != previous era).
            Uses era_progress and transition_seconds to compute blend_factor.
            """
            # Implementation: track previous era, detect change, compute blend

        def begin_outgoing(self) -> None:
            """Bind transition FBO for rendering the outgoing era."""
            self.transition_fbo.use()
            self.transition_fbo.clear(0.0, 0.0, 0.0, 0.0)

        def composite(self, incoming_texture: moderngl.Texture,
                      target_fbo) -> None:
            """Composite outgoing + incoming textures with blend factor."""
            target_fbo.use()
            self.transition_texture.use(location=0)
            incoming_texture.use(location=1)
            self.composite_prog["u_outgoing"].value = 0
            self.composite_prog["u_incoming"].value = 1
            self.composite_prog["u_blend_factor"].value = self.blend_factor
            self.quad.render(self.composite_prog)

        def resize(self, width, height): ...
        def release(self): ...
    ```

    The transition logic:
    - Track self._prev_era (initialized to -1).
    - When current_era changes from _prev_era, set in_transition=True, outgoing_era=_prev_era, incoming_era=current_era, transition_elapsed=0.0, transition_duration from incoming era's EraVisualConfig.
    - Each frame during transition: transition_elapsed += frame_time. blend_factor = smoothstep(clamp(transition_elapsed / transition_duration, 0.0, 1.0)). Where smoothstep(t) = t*t*(3-2*t).
    - When blend_factor >= 1.0: in_transition=False, _prev_era=current_era.
    - When NOT in transition and era has not changed: just update _prev_era.

    **In __init__:**
    - Create EraTransitionManager: `self.transition = EraTransitionManager(self.ctx, self.wnd.size[0], self.wnd.size[1])`
    - Pre-compute physics lookup tables at startup (NOT per-frame):

    **In on_render -- New render pipeline:**
    1. Update simulation -> PhysicsState (unchanged)
    2. Update camera (unchanged)
    3. Look up current era's visual config: `config = get_era_visual_config(state.current_era)`
    4. Update post-processing bloom params from era config:
       `self.postfx.bloom_strength = config.bloom_strength`
       `self.postfx.bloom_threshold = config.bloom_threshold`
    5. Compute era-specific physics uniforms:
       ```python
       # Physics uniform values (only computed for eras that use them)
       physics_uniforms = {}
       if state.current_era == 5:  # Nucleosynthesis
           bbn = get_bbn_fractions(state.temperature)
           physics_uniforms['u_helium_fraction'] = bbn['helium_fraction']
       elif state.current_era == 6:  # Recombination
           physics_uniforms['u_ionization_fraction'] = get_ionization_fraction(
               state.temperature, self.ionization_table)
       elif state.current_era == 8:  # First Stars
           # Reionization fraction: ramp from 0 to 1 over the era
           physics_uniforms['u_reionization_frac'] = state.era_progress
       elif state.current_era in (9, 10):  # Galaxy Formation, LSS
           physics_uniforms['u_collapsed_fraction'] = get_collapsed_fraction(
               state.cosmic_time, self.collapsed_fraction_table)
       ```
    6. Upload per-era compute uniforms before dispatch:
       ```python
       self.particles.compute["u_expansion_rate"].value = config.expansion_rate
       self.particles.compute["u_noise_strength"].value = config.noise_strength
       self.particles.compute["u_gravity_strength"].value = config.gravity_strength
       self.particles.compute["u_damping"].value = config.damping
       ```
    7. Update particle system (compute dispatch) -- same as before
    8. Switch to current era's shader: `self.particles.set_era_shader(state.current_era)`
    9. Check for era transition: `self.transition.check_transition(state.current_era, state.era_progress, config.transition_seconds)`

    **Rendering with transitions:**
    If self.transition.in_transition:
      a. Render OUTGOING era:
         - self.transition.begin_outgoing() -- binds transition FBO
         - Set outgoing era's shader: self.particles.render_with_shader_key(outgoing_config.shader_key, proj_bytes, view_bytes)
         - Upload outgoing config uniforms before render
      b. Render INCOMING era into HDR FBO:
         - self.postfx.begin_scene() -- binds HDR FBO
         - Enable blend, depth settings (same as current)
         - Upload incoming (current) era's uniforms including physics_uniforms
         - self.particles.render(proj_bytes, view_bytes) -- renders with current era shader
         - Restore blend/depth state
      c. Composite: self.transition.composite(self.postfx.hdr_texture, self.postfx.hdr_fbo)
         This writes the blended result back into hdr_fbo
      d. self.postfx.end_scene() -- runs bloom + tonemap on the composited result

    If NOT in transition (normal path):
      a. self.postfx.begin_scene()
      b. Enable blend, depth settings
      c. Upload current era uniforms + physics uniforms
      d. self.particles.render(proj_bytes, view_bytes)
      e. Restore state
      f. self.postfx.end_scene()
