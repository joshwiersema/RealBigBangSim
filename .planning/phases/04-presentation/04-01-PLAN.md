    1. A `MilestoneNotification` dataclass:
       - `milestone: Milestone` (the milestone that triggered)
       - `time_remaining: float` (seconds until notification disappears)
       - `total_duration: float` (total display time, for alpha calculation)

    2. A `MilestoneManager` class:
       - `__init__(self, milestones: list[Milestone], display_duration: float = 5.0, fade_duration: float = 1.0)`:
         Store milestones sorted by cosmic_time. Initialize a `_triggered` list of booleans (all False). Initialize an empty `_active: list[MilestoneNotification]`.
         Store `_next_index: int = 0` as optimization pointer (Pitfall 4: use sorted list with monotonically advancing index, not linear scan).

       - `update(self, cosmic_time: float, dt: float) -> list[Milestone]`:
         Advance the index pointer: while `_next_index < len(milestones)` and `cosmic_time >= milestones[_next_index].cosmic_time`, fire the milestone (append to `_active` as MilestoneNotification, mark `_triggered[_next_index] = True`, increment `_next_index`). Return list of newly triggered milestones.
         Also tick down `time_remaining` on all active notifications by `dt`, removing any that reach <= 0.

       - `get_active_notifications(self) -> list[MilestoneNotification]`:
         Return list of currently active (non-expired) notifications.

       - `get_notification_alpha(self, notif: MilestoneNotification) -> float`:
         Returns alpha (0.0-1.0) for rendering. Full opacity during display, linear fade from 1.0 to 0.0 during the last `fade_duration` seconds. Use: if `time_remaining <= fade_duration`: return `time_remaining / fade_duration`, else return 1.0.

       - `reset(self)`:
         Clear all triggered flags, reset `_next_index` to 0, clear active notifications.

       - `@property triggered_count(self) -> int`:
         Return number of milestones triggered so far.
