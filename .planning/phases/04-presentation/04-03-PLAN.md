    **HUDManager class:**

    ```python
    class HUDManager:
        def __init__(self):
            self.visible: bool = True
            self.show_physics: bool = True
            self.show_education: bool = True
            self.show_milestones: bool = True
    ```

    **render(self, state: PhysicsState, sim: SimulationEngine, milestones: MilestoneManager, camera_auto: bool, eras: list) -> None:**
    If not self.visible, return immediately.
    Get display_size from imgui.get_io().display_size.
    Call each panel method in sequence:
    1. `_render_era_panel(state, eras, display_size)` -- top-left
    2. `_render_physics_panel(state, display_size)` -- top-right (if show_physics)
    3. `_render_education_panel(state, display_size)` -- left side below era panel (if show_education)
    4. `_render_milestone_notifications(milestones, display_size)` -- center-top (if show_milestones)
    5. `_render_timeline_bar(state, sim, eras, display_size)` -- bottom
    6. `_render_controls_hint(camera_auto, display_size)` -- bottom-right corner

    **toggle(self):** Flip self.visible.

    **Panel method details:**

    `_render_era_panel(self, state, eras, display_size):`
    Position: top-left (20, 20). Background alpha 0.6.
    Content: Era name in warm color (1.0, 0.9, 0.7) using set_window_font_scale(1.5) for header, then era description from the EraDefinition.description (the short one-liner).

    `_render_physics_panel(self, state, display_size):`
    Position: top-right (display_size.x - 280, 20). Background alpha 0.6.
    Content: Title "Physics" then:
    - Temperature: format_physics_value(state.temperature, "K")
    - Scale Factor: f"{state.scale_factor:.6e}"
    - Matter Density: format_physics_value(state.matter_density, "kg/m^3")  -- use Unicode superscript or just kg/m^3
    - Radiation Density: format_physics_value(state.radiation_density, "kg/m^3")
    - Hubble Parameter: format_physics_value(state.hubble_param, "km/s/Mpc")
    - Cosmic Time: format_cosmic_time(state.cosmic_time) -- human-readable (e.g., "3.8e+05 years" or "180 seconds")

    Helper `format_physics_value(value: float, unit: str) -> str`:
    If abs(value) < 1e-3 or abs(value) > 1e6: return f"{value:.2e} {unit}"
    Else: return f"{value:.4f} {unit}"

    Helper `format_cosmic_time(seconds: float) -> str`:
    If seconds < 60: return f"{seconds:.2e} s"
    If seconds < 3600: return f"{seconds/60:.1f} min"
    If seconds < 86400 * 365.25: return f"{seconds/3600:.1f} hr"
    If seconds < 86400 * 365.25 * 1e6: return f"{seconds/(86400*365.25):.2e} yr"
    If seconds < 86400 * 365.25 * 1e9: return f"{seconds/(86400*365.25*1e6):.2f} Myr"
    Else: return f"{seconds/(86400*365.25*1e9):.2f} Gyr"

    `_render_education_panel(self, state, display_size):`
    Position: left side (20, 120). Background alpha 0.5. Max width ~350px.
    Content: ERA_DESCRIPTIONS[state.current_era] text (the rich 2-4 sentence version from educational_content.py). Use imgui.push_text_wrap_pos() for word wrapping.

    `_render_milestone_notifications(self, milestones, display_size):`
    For each notification in milestones.get_active_notifications():
    Position: center-top, stacked vertically. Each notification is a small window.
    Alpha from milestones.get_notification_alpha(notif).
    Content: Milestone name (header scaled 1.3x), milestone description below.
    Use imgui.set_next_window_bg_alpha(0.8 * alpha) for fade effect.
    Unique window ID per notification: f"Milestone##{notif.milestone.name}" to prevent imgui ID conflicts.

    `_render_timeline_bar(self, state, sim, eras, display_size):`
    Uses imgui foreground draw list (imgui.get_foreground_draw_list()) to draw directly on screen.
    Bar dimensions: height=30, y = display_size.y - 40, x from 20 to display_size.x - 20.
    For each era: draw a filled rect proportional to its screen_seconds / total_screen_seconds, colored by ERA_COLORS[i].
    Draw era name labels on segments wide enough (>40px).
    Draw progress indicator: white vertical line at current position.
    Compute overall progress: sim.screen_time / sim.timeline.total_duration().

    `_render_controls_hint(self, camera_auto, display_size):`
    Position: bottom-right (display_size.x - 220, display_size.y - 70). Background alpha 0.4.
    Content: "H: Toggle HUD | C: Camera [Auto/Free] | Space: Pause | +/-: Speed"
