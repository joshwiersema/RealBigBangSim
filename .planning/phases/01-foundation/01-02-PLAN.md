    **Step 2: Create `bigbangsim/simulation/cosmology.py`:**

    This module pre-computes a dense lookup table of cosmological parameters at startup by integrating the Friedmann equation.

    Class `CosmologyModel`:
    - `__init__(self)` -- At construction, integrate the Friedmann equation from a_initial to a=1.0 using `scipy.integrate.solve_ivp(method='Radau')` to get scale factor as a function of cosmic time. Store results as a dense lookup table (NumPy arrays).
    - Key arrays: `_cosmic_times: np.ndarray`, `_scale_factors: np.ndarray`, `_temperatures: np.ndarray`, `_matter_densities: np.ndarray`, `_radiation_densities: np.ndarray`, `_hubble_params: np.ndarray`

    The Friedmann equation to integrate:
    ```
    H(a) = H0 * sqrt(Omega_r0 / a^4 + Omega_m0 / a^3 + Omega_Lambda0)
    dt/da = 1 / (a * H(a))
    ```

    Integration approach:
    - Integrate `dt/da` from a_start (very small, e.g., 1e-12) to a=1.0
    - Use `solve_ivp` with method `'Radau'` (implicit solver for stiff equations at radiation-matter transition)
    - Generate ~10,000 sample points using `dense_output=True`
    - For each scale factor a, compute: T(a) = T_CMB0 / a, rho_m(a) = rho_crit * Omega_m0 / a^3, rho_r(a) = rho_crit * Omega_r0 / a^4
    - rho_crit = 3 * H0_SI^2 / (8 * pi * G)

    Methods:
    - `get_state_at_cosmic_time(self, cosmic_time: float) -> dict` -- interpolates lookup table to return scale_factor, temperature, matter_density, radiation_density, hubble_param at the given cosmic time. Uses `np.interp` for O(1) lookup.
    - `get_state_at_scale_factor(self, a: float) -> dict` -- same but keyed by scale factor.

    NOTE: For very early times (before a_start), extrapolate using radiation-dominated approximation: a(t) ~ sqrt(2 * H0_SI * sqrt(Omega_r0) * t), T = T_CMB0 / a(t).

    **Step 3: Create `bigbangsim/simulation/engine.py`:**

    Class `SimulationEngine` implementing the "Fix Your Timestep!" accumulator pattern:

    ```python
    from copy import copy
    from bigbangsim.config import PHYSICS_DT, DEFAULT_SPEED, MIN_SPEED, MAX_SPEED
    from bigbangsim.simulation.state import PhysicsState
    from bigbangsim.simulation.timeline import TimelineController
    from bigbangsim.simulation.cosmology import CosmologyModel
    from bigbangsim.simulation.eras import ERAS

    class SimulationEngine:
        def __init__(self):
            self.timeline = TimelineController(ERAS)
            self.cosmology = CosmologyModel()
            self.accumulator = 0.0
            self.screen_time = 0.0  # Wall-clock playback position
            self.speed_multiplier = DEFAULT_SPEED
            self.paused = False
            self.state = self._compute_state(0.0)
            self.prev_state = copy(self.state)

        def _compute_state(self, screen_time: float) -> PhysicsState:
            """Compute full physics state from screen time."""
            cosmic_time = self.timeline.screen_to_cosmic(screen_time)
            era_idx, era_progress = self.timeline.get_current_era(cosmic_time)
            params = self.cosmology.get_state_at_cosmic_time(cosmic_time)
            return PhysicsState(
                cosmic_time=cosmic_time,
                scale_factor=params['scale_factor'],
                temperature=params['temperature'],
                matter_density=params['matter_density'],
                radiation_density=params['radiation_density'],
                hubble_param=params['hubble_param'],
                current_era=era_idx,
                era_progress=era_progress,
            )

        def update(self, frametime: float) -> tuple[PhysicsState, float]:
            """Advance simulation. Returns (interpolated_state, alpha)."""
            if self.paused:
                return self.state, 0.0

            self.accumulator += frametime * self.speed_multiplier

            while self.accumulator >= PHYSICS_DT:
                self.prev_state = copy(self.state)
                self.screen_time += PHYSICS_DT
                self.screen_time = min(self.screen_time, self.timeline.total_duration())
                self.state = self._compute_state(self.screen_time)
                self.accumulator -= PHYSICS_DT

            alpha = self.accumulator / PHYSICS_DT
            return self.prev_state.lerp(self.state, alpha), alpha

        def toggle_pause(self):
            self.paused = not self.paused

        def set_speed(self, multiplier: float):
            self.speed_multiplier = max(MIN_SPEED, min(MAX_SPEED, multiplier))

        def increase_speed(self):
            self.set_speed(self.speed_multiplier * 2.0)

        def decrease_speed(self):
            self.set_speed(self.speed_multiplier / 2.0)
    ```
